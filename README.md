Данный скрипт гененрирует карту подшпиля для Gems of War
Нужно с помощью любого https анализатора получить ответ от сервера
ПОсле запкуска игры открываем подшпиль и находим ответ на запрос "functionName": "live_event_get_info"
Копируем тело ответа на страничку и получаем карту


ПРойденные комнаты  можгут отображаться не корртектно, т.к. в некоторых случаях есть переопределение координат
