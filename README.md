# 🗺️ Underspire Map Live — Карта Подшпиля для Gems of War

Этот инструмент позволяет **визуализировать вашу текущую карту Подшпиля (Underspire)** прямо в браузере — без установки, без сервера, полностью на клиенте.  
Подходит для игроков **Gems of War**, кто хочет отслеживать пройденные комнаты, боссов, сокровища и ворота в режиме реального времени.

---

## 🚀 Как использовать

1. **Запустите Gems of War** и откройте Подшпиль (Underspire).
2. Откройте **инструменты разработчика** в браузере (F12 → вкладка **Network** / Сеть).
3. Найдите сетевой запрос, содержащий:
   ```json
   "functionName": "live_event_get_info"
   ```
4. Скопируйте тело ответа от сервера (вкладка Response).
5. Перейдите на страницу:
👉 https://anch665.github.io/GoW-Underspire-map/
6. Вставьте JSON в поле «Вставить JSON» → нажмите «Загрузить». 💡 Также можно просто перетащить JSON-файл в область загрузки.
Готово! Вы увидите интерактивную карту 20×20 с вашим прогрессом.
